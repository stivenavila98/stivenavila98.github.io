Descripcion

Esta es una aplicación web la cual desarrolle junto a un equipo para una Universidad de Maracaibo Venezuela llamada URBE.

La aplicación fue creada con la función de controlar de manera remota un gran número de PCs de forma simultánea, entre las principales opciones que posee está la del encendido de un grupo seleccionado de PCs como también incluye las opciones de reiniciar y apagar.

Posee un sistema que le permite mostrar diferentes opciones dependiendo de los privilegios que posea el usuario con el ingresó a la aplicación, si es un usuario estándar el sistema no permite controlar PCs que estén fuera del área en la que se está ejecutando, mientras que un usuario con todos los privilegios puede pasar por encima de esos y otros límites.

Posee un panel el cual permite colocar a las PCs seleccionadas en un estado de reservación.

¿Que es el estado de reservación?

Es el modo en el cual las PCs tienen un límite de tiempo para su uso (Ideal para Universidades y Liceos en los que hay un gran número de estudiantes y todos quieren hacer uso de una PC). Este sistema permite hacer una reservación con simplemente la cedula de identidad y al culminar el tiempo de uso dicha PC se pone en modo disponible para que un estudiante en espera pueda hacer uso de la PC.

El sistema permite enviar mensajes en forma de alerta a la PC seleccionada (Excelente para esos momentos en los que quieres informarle al estudiante que baje el tono de voz ya que pude desconcentrar a otros que estén haciendo uso de las PCs).
